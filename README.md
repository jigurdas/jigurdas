## Автор Github профіля Jigurdas
Мене звати **Jigurdas** я займаюсь Game development, Python development, C# development с 2021 року.

В 2022 році заснувував компанію Juggers. Делюсь своими знаниями в области программирования и создания сайтов. Сформировал свою собственную методику обучения, которая позволяет ученикам максимально эффективно осваивать изучаемый материал. Огромное количество моих учеников уже научились создавать сайты самостоятельно и зарабатывать на этом деньги.

Вивчаю мови C#, C++/C, Python, системой контроля версій Git,  а ще багато іншого із світу веб-технологій.
